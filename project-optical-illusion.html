GENERATIVE ART 005

Optical Illusion

A motion-based illusion that plays with repetition and pattern to distort visual perception.

reflection

I wanted to challenge myself a bit with this assignment by exploring the different shapes, patterns and level of detail I can create in P5. During the last assignment I discovered how I can use bezier curves to create shapes and patterns, so for this assignment I wanted to experiment more with the bezier function to create more complex shapes. I quickly discovered that while the function can be very useful for creating simples curved forms, the more detailed and elaborate the patterns become, the more difficult and time cosuming it becomes to control the curves using the coordinates. I definetly think I was a bit too ambitious in the pattern I chose for the optical illusion and as a result I had to simplify the original design. Because I spent so much time on the pattern, I didn't get the chance to actually animate the sketch like I intended and instead ended up using the mouse pressed function to achieve a similar effect instead. Given the chance to go this again, I would definetly go with a simpler pattern so I actually have time to animate the sketch.
