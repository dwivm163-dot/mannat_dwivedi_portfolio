GENERATIVE ART 004

Exquisite Corpse

A surreal composite figure constructed collaboratively through layered generative drawing.

reflection

final figure backstory: the product of a mad scientists experiment, part cyclops, part dragon. reflection: I had a really hard time in the start because I wanted to try using bezier curves to create the figure but sinse I've never used it before and its not something we've covered in class yet I struggled a lot but once I figured it out I actually found it to be much easier to use than arcs and ellipse'. Im glad I used the oppurtunity to challenge myself a bit since I feel a lot more confident with using different functions on p5JS. This assignment was really fun and I'm looking forward to working on more collaborative projects like this in the future.
